📝 Sticky Notes Web App

A simple and interactive Sticky Notes application built using HTML, CSS, and JavaScript. This app allows users to create, edit, and delete notes, with automatic saving using the browser's local storage.


🚀 Features

- ➕ Add new sticky notes
- ✏️ Edit notes in real-time
- ❌ Delete notes easily
- 💾 Auto-save notes using localStorage
- 📱 Responsive and clean UI


📁 Project Structure

sticky-notes/
│── index.html
│── style.css
│── script.js


🧩 Technologies Used

- HTML5 – Structure of the app
- CSS3 – Styling and layout
- JavaScript (Vanilla JS) – Functionality and logic
- Local Storage – Data persistence


▶️ How to Run

1. Download or clone the project
2. Open the project folder
3. Run the "index.html" file in any web browser


⚙️ How It Works

- Clicking “Add Note” creates a new sticky note
- Notes can be edited directly inside the text area
- Changes are saved automatically
- Clicking the “×” button deletes a note
- All notes are stored in the browser’s local storage and persist after refresh
